When coding colors for your design, there are a few different ways of denoting the shade.

The easiest, but most limited, method is to simply type any one of the 140 possible color names supported by all browsers.

CSS Example:

color: blue

For a more powerful use of color, you can use any of the following methods to select from over 16 million different colors.

Along with color, we can use font families to decide how our typeface actually looks. There are plenty of different font families out there but they all typically reside under three different categories:

  • Serif
  • Sans-Serif
  • Monospace

Serifs are the small lines attached to letters. Their origins are a mystery; one theory suggests they arose when scribes using brushes or quills left small marks with the writing implement as they finished each stroke. This evolved into deliberately adding smaller strokes in more regular, artful ways, and those decorative strokes became an expected part of the letters.
Source

A sans-serif letterform is one that does not have extending features called "serifs" at the end of strokes. Sans-serif typefaces tend to have less stroke width variation than serif typefaces. They are often used to convey simplicity and modernity or minimalism.
Source

A monospaced font, also called a fixed-pitch, fixed-width, or non-proportional font, is a font whose letters and characters each occupy the same amount of horizontal space. This contrasts with variable-width fonts, where the letters and spacings have different widths.
Source

Text Style

H1: Main Page Heading

  • font-weight: 700
  • font-size: 36px
  • font-family: Ubuntu

The H1 header is typically used for the main heading at the top of a page. For semantic purposes, remember to only use the H1 header once within a webpage as there is usually only one main heading per page.

H2: Subheading

  • text-align: left
  • margin-left: 0
  • font-family: Ubuntu

An H2 header is a secondary header that you can use to emphasize secondary keywords, break up content to make it scannable and easy to read, highlight important pieces of information, and saturate content with keywords.

Source

P: Paragraph Text

  • font-size: 20px
  • font-family: Ubuntu
  • font-weight: 400

The <p> parapraph tag is used for all normal text throughout a webpage. Use this tag for descriptions, paragraphs, and general information.
